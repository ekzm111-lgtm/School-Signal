# 🏫 시험 시간 안내 자동 송출 시스템 (Supertonic Broadcast)

본 시스템은 하이브 수퍼톤(Supertone)의 초경량 온디바이스 TTS 모델인 **Supertonic**을 활용하여, 학교 시험 도중 종료 N분 전 및 종료 시각에 맞춰 안내 방송을 로컬 스피커로 자동 송출해 주는 통합 제어 시스템입니다.

---

## 🚀 1. 시스템 구동 방법 (관리자용)

### 💻 서버 PC 설정 및 실행
1. **스피커/앰프 연결**: 서버 프로그램이 구동되는 PC의 오디오 출력 단자(헤드폰 잭 등)를 학교 방송실의 방송 앰프 입력 장치에 연결합니다.
2. **의존성 라이브러리 설치** (최초 1회):
   ```powershell
   pip install -r requirements.txt
   ```
3. **서버 실행**:
   프로젝트 폴더에서 아래 명령어를 실행하여 서버를 가동합니다.
   ```powershell
   python app.py
   ```
   *(※ 최초 구동 시 AI 음성 합성 모델 파일(약 400MB)이 Hugging Face를 통해 자동으로 로컬에 다운로드됩니다.)*

---

## 📱 2. 시스템 접속 방법

서버가 실행되면 아래의 주소를 통해 웹 제어판 대시보드에 접속할 수 있습니다.

### A. 동일 유선망 PC에서 접속할 때
*   👉 **[http://127.0.0.1:8000](http://127.0.0.1:8000)**

### B. 휴대폰(모바일) 또는 타 네트워크 망에서 원격 접속할 때
학교 내부 보안 정책상 유/무선망이 격리되어 있더라도 아래의 우회 인터넷 주소로 접속할 수 있습니다.
*   👉 **[https://tough-olives-battle.loca.lt](https://tough-olives-battle.loca.lt)**
*   **접속 시 최초 1회 인증 절차**: 
    1. 화면에 나오는 `This tunnel is hosted by: 59.8.0.131` 표시를 확인합니다.
    2. 입력란(`IP Address:`)에 **`59.8.0.131`**을 입력합니다.
    3. 파란색 **[Continue]** 버튼을 클릭하면 제어판 화면으로 진입합니다.

## 🛠️ 3. 주요 기능 사용 방법

### ① 일과 시정표 빠른 선택
*   '시험 일정 등록' 카드 상단의 **[일과 시정표 빠른 선택]** 드롭다운에서 당일 시험을 치를 교시를 선택합니다.
*   교시를 클릭하면 해당 교시의 실제 종료 시각(예: 3교시는 `11:25`)이 자동 입력되며, 과목명 칸에도 `3교시 ` 라는 텍스트가 접두사로 자동 완성됩니다.
*   과목명 칸에 영어, 수학 등 실제 시험 과목명을 이어 쓰시면 됩니다.

### ② 방송 일정 예약 등록
*   교시와 과목명, 종료 시각을 확인한 후 **송출할 시점**(10분 전, 5분 전, 1분 전, 종료 시점 등)을 체크합니다.
*   **[방송 일정 예약 등록]** 버튼을 누르면 시스템 내부에서 AI 목소리로 안내 멘트를 미리 소리 파일(.wav)로 생성하여 대기합니다.
*   등록이 완료되면 우측 **'예정된 방송 목록'**에 등록되고, 좌측 상단 원형 타이머에 다음 방송까지의 남은 시간이 **실시간 카운트다운**됩니다.
*   정해진 시간이 되면 서버 PC 스피커를 통해 방송이 정밀한 타이밍에 자동 송출됩니다.

### ③ 즉시 마이크 방송 (수동 즉시 송출)
*   갑작스러운 공지사항이나 돌발 방송이 필요할 때 사용합니다.
*   텍스트창에 방송할 내용을 직접 입력한 후, 목소리 스타일을 선택합니다.
*   **[미리듣기]**를 클릭하면 휴대폰이나 관리자 PC 스피커로 오디오를 미리 들으며 문장을 다듬을 수 있습니다.
*   **[즉시 송출]**을 클릭하면 백그라운드 스레드를 통해 곧바로 서버 PC 및 학교 방송망으로 음성이 송출됩니다.

### ④ 일정 관리 및 송출 기록 모니터링
*   **일정 삭제**: 시험이 취소되었거나 수정이 필요할 경우, 우측 일정 목록에서 빨간색 **[삭제]** 버튼을 누르면 예약된 방송 일정이 즉각 취소되며 생성된 오디오 파일도 자동 정리됩니다.
*   **기록 확인**: 하단의 **'송출 기록 로그'**를 통해 실제로 방송이 성공적으로 나갔는지 실시간으로 성공/실패 여부를 모니터링할 수 있습니다.

## 📢 4. 기본 방송 멘트 안내

시스템이 자동으로 합성하는 기본 멘트 구조는 아래와 같습니다.

*   **종료 N분 전 방송 (예: 5분 전)**:
    > "아 아, 시험 종료 안내 말씀 드립니다. 잠시 후 [등록한 시험 과목명] 시험 종료 5분 전입니다. 수험생 여러분께서는 답안 마킹을 다시 한번 확인하시기 바랍니다."
*   **시험 종료 정시 방송**:
    > "아 아, 시험 종료 안내 말씀 드립니다. [등록한 시험 과목명] 시험 시간이 모두 종료되었습니다. 수험생 여러분께서는 필기도구를 내려놓으시고 답안지를 제출해 주시기 바랍니다."

※ 기본 멘트 외에 다르게 방송하고 싶으신 경우, 예약 등록 시 **[멘트 개별 설정]** 메뉴를 열어 칸마다 원하시는 텍스트를 커스텀하여 직접 등록할 수 있습니다.
